# VideoFoundry

## Mission

Runtime and content-pipeline consumer surface that builds on the ProtocolWarden
ecosystem rather than redefining it.

## This Repo Is

- runtime consumer
- content pipeline surface
- backend-adjacent integration point
- consumer of normalized contracts and public-safe projections

## This Repo Is Not

- canonical graph language authority
- routing policy owner
- orchestration authority
- public projection authority

## Depends On

- `OperationsCenter`
- `SwitchBoard`
- `ExecutorRuntime`
- `Warehouse`
- `PlatformDeployment`

## Boundary note

VideoFoundry can be documented publicly without being treated as a semantic
authority. The public site should explain its role, not promote it into graph
ownership.
