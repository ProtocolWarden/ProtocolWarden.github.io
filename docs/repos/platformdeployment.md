# PlatformDeployment

## Mission

Runtime glue and deployment/topography overlay support for the
ProtocolWarden ecosystem.

## This Repo Is

- local deployment and hosting layer
- machine-specific topography and runtime placement surface
- environment assembly support
- convenience wrapper host for boundary-artifact-driven Custodian runs

## This Repo Is Not

- orchestration engine
- graph language owner
- ontology owner
- projection owner
- deployment policy authority
