# RepoGraph

## Mission

Canonical shared graph-semantics library for the ProtocolWarden platform.

## This Repo Is

- canonical graph language authority
- schema governance owner
- projection and boundary semantics owner
- shared vocabulary for repo identity, visibility, disclosure, and topology

## This Repo Is Not

- private graph truth source
- runtime orchestrator
- deployment manager
- UI projection engine
- policy blob

## Depends On

- none for canonical semantics
- `Custodian` for enforcement
- `PlatformManifest` for public projection publication
- `private-truth layer` for private truth export
