# PlatformDeployment (`PlatformDeployment`)

## Mission

Deployment, hosting, and local topography support.

## This Repo Is

- the current repository identity for `PlatformDeployment`
- local deployment and hosting layer
- machine-specific topography and runtime placement surface
- service and environment management support

## This Repo Is Not

- orchestration engine
- topology owner
- protocol contract owner
- public platform language owner
- private topology truth owner

This page is historical. It documents the `PlatformDeployment` repo identity
after the `WorkStation` rename and should not be treated as the canonical
architecture source.
