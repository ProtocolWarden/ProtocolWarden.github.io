# External Integrations

This page groups repos that are relevant to the ecosystem but are not core
ProtocolWarden semantic authority.

## Categories

- Forked dependency
- Adapter candidate
- Backend candidate
- External capability source

## Examples

- Archon
- firecrawl
- kodo
- PraisonAI
- Wan2GP
- openclaw
- Zonos-API
- neuro-game-sdk

## Rule

These repos are not equal to core platform repos. They only become platform
components when the contract and boundary story is explicit.
