# Glossary

## Ontology

What exists and what it means.

## Topology

How system elements connect.

## Topography

The operational terrain: hotspots, instability, and governance pressure.

## Projection

A derived public-safe or otherwise scoped view from a richer source of truth.

## Governance

The rules and enforcement surfaces that keep architecture and operations within
acceptable boundaries.

## Runtime

The execution mechanics and environment where work actually runs.

## Orchestration

The planning, routing, and coordination layer above raw runtime execution.
