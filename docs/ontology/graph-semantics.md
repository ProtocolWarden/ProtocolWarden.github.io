# Graph Semantics

RepoGraph defines the shared graph language for ProtocolWarden.

## It owns

- repo identity semantics
- visibility and disclosure semantics
- edge and topology semantics
- projection and redaction semantics
- boundary artifact schema semantics
- schema versioning and compatibility policy

## It does not own

- private graph truth
- runtime orchestration
- deployment control
- UI projection logic
- policy governance blobs
