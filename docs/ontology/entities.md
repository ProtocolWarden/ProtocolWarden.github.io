# Entities

This section tracks the main ecosystem entity types.

## Examples

- Project
- Repository
- ManagedProject
- ManagedRepository
- Manifest
- Artifact
- Run
- Audit
- Evidence
- VisibilityPolicy
- ProjectionRule
- RuntimeBinding
- CapabilitySet

For each entity, the important questions are:

- what is it
- who owns its semantics
- where does it live
- how does it relate to other entities
